
# Privacy Policy (Draft)

## Data Collected
- Name
- Phone number
- Email
- Address
- ID images (front/back)

## Purpose
Verification to ensure safety and prevent fraud.

## Storage & Retention
- Images deleted within 30–90 days
- Only verification status retained

## Security
- Encrypted storage
- HTTPS in transit
- Restricted admin access

## User Rights
- Access
- Deletion
- Correction

Complies with Texas Data Privacy and Security Act (TDPSA).
